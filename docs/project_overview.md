# Coursera Guided Project Overview

## Course Title
**Python for Data Visualization: Matplotlib & Seaborn**

## Instructor
Ryan Ahmed

## Platform
Coursera — Guided Project


---

## Course Description

This guided project introduces the fundamentals of data visualization using Python. It focuses on two widely used libraries — **Matplotlib** and **Seaborn** — and demonstrates how to create meaningful visual representations of data.

The course emphasizes hands-on learning through step-by-step tasks completed in a cloud-based Jupyter environment.

---

## Skills Practiced

- Python programming fundamentals
- Data visualization techniques
- Statistical visualization
- Plot customization and styling
- Exploratory data analysis

---

## Tasks Completed

1. Plot basic line charts
2. Create scatter plots
3. Build pie charts
4. Generate histograms
5. Plot multiple variables on a single figure
6. Create subplot layouts
7. Visualize data using Seaborn
8. Explore pair plots, distributions, and heatmaps

## Datasets Used

- Stock price and daily return datasets
- Breast cancer dataset from `sklearn.datasets`
